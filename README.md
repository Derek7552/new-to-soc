# New to SOC - SOC 新人学习指南

面向安全运营中心（Security Operations Center）新人的系统化学习路径与知识库。

## 目录结构

```
docs/
  01-fundamentals/    # SOC 基础概念
  02-tools/           # 常用工具指南
  03-processes/       # 运营流程与规范
  04-skills/          # 技术能力提升
  05-resources/       # 学习资源汇总
```

## 适用对象

- 即将加入 SOC 团队的新人
- 希望转型安全运营方向的 IT 从业者
- 对安全运营感兴趣的在校学生

## 如何使用

按照 `01` 到 `05` 的顺序阅读，建立系统化的 SOC 知识体系。每个模块可独立学习，但建议先完成基础概念部分。
