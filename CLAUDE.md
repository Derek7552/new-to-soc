# CLAUDE.md

## Project Overview

"New to SOC" 是一个面向 SOC（安全运营中心）新人的纯 Markdown 学习指南项目。

## Structure

- `docs/01-fundamentals/` - SOC 基础概念（CIA 三要素、攻击链、威胁情报等）
- `docs/02-tools/` - 常用安全工具（SIEM、EDR、Wireshark 等）
- `docs/03-processes/` - 运营流程（告警分诊、事件响应、值班流程等）
- `docs/04-skills/` - 技术能力（日志分析、网络分析、恶意软件基础等）
- `docs/05-resources/` - 学习资源（书籍、认证、实验平台、社区等）

## Writing Conventions

- 语言：中文为主，专业术语保留英文并在首次出现时标注中文释义
- 文件命名：小写英文 + 连字符，如 `incident-response.md`
- 每篇文档开头用一句话说明本篇的学习目标
- 实操内容优先：理论配合实际场景或案例
- 避免过度堆砌链接，注重内容本身的完整性
